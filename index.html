Welcome to the robocode-pmj-dacruzer project

This project implements two branches providing:

  • A simple project (on the master branch) with the minimal POM file for building and running robocode robots.
  • A more complex project (on the quality-assurance branch) with a more extensive POM file that includes support for PMD, FindBugs, and Checkstyle.
